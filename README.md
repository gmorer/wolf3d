# Wolf3D

## Dependencies

- SDL2

### - Linux

```sh
make install
make linux
```

### - Macos

```sh
make install
make linux
```

## Installation

```sh
git clone https://github.com/gmorer/SDL2.git && cd SDL2 && make install && make
```

## Usage

```sh
./wolf3d [map]
```
